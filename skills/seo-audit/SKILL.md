# SEO审计与优化（seo-audit）

## 描述

SEO审计与优化 - 审计、诊断和优化网站的SEO问题，提升搜索引擎排名。分析技术SEO、内容SEO和外链情况，提供具体的改进建议。

## 功能

- 技术SEO审计：检查网站技术层面的SEO问题
- 内容SEO分析：分析内容的关键词优化情况
- 外链分析：评估外链质量和数量
- 竞争对手分析：分析竞争对手的SEO策略
- 关键词研究：发现和优化目标关键词
- SEO改进建议：提供可执行的优化方案

## 审计维度

### 1. 技术SEO

- 网站速度和性能
- 移动端适配
- 索引和抓取问题
- 网站架构和URL结构
- 重复内容问题
- 重定向和404错误

### 2. 页面SEO

- 标题标签优化
- Meta描述
- 标题层级结构（H1-H6）
- 图片Alt标签
- 内部链接结构
- Schema标记

### 3. 内容SEO

- 关键词密度和分布
- 内容质量和原创性
- 内容更新频率
- 长尾关键词覆盖
- 用户意图匹配

### 4. 外链分析

- 外链数量和质量
- 外链来源多样性
- 锚文本分布
- 垃圾外链识别

## 使用场景

- **网站上线前**：确保新网站SEO基础完善
- **排名下降**：诊断排名下降的原因
- **定期维护**：定期检查SEO健康状况
- **竞争对手分析**：了解竞争对手的SEO策略
- **内容优化**：优化现有内容的SEO表现
