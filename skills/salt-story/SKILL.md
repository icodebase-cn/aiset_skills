# 盐选故事写作（salt-story）

## 描述

盐选故事写作 - 生成有反转、有吸引力、无 AI 味的知乎盐选付费故事。专为知乎盐选专栏设计的创意写作技能，擅长构建悬念、制造反转、塑造人物。

## 功能

- 故事构思：基于主题生成完整的故事大纲
- 人物塑造：创建有深度、有冲突的人物角色
- 情节设计：设计引人入胜的情节和反转
- 开头吸引：撰写能留住读者的开篇
- 悬念构建：在故事中设置合理的悬念和伏笔
- 结尾反转：设计令人意想不到的结尾

## 盐选故事特点

### 1. 开篇要抓人

- 第一人称视角，增强代入感
- 开头即冲突，不拖沓铺垫
- 制造信息差，让读者好奇
- 金句开场，引发共鸣

### 2. 节奏要紧凑

- 情节密度高，每段都有推进
- 对话简洁有力，推动剧情
- 场景切换快，保持紧张感
- 适当留白，给读者想象空间

### 3. 人物要立体

- 主角有缺陷，更真实可信
- 反派有动机，不脸谱化
- 人物关系复杂，有张力
- 情感变化有层次

### 4. 反转要自然

- 伏笔提前埋，反转有依据
- 多重反转，层层递进
- 既在意料之外，又在情理之中
- 结尾留余韵，让人回味

## 写作流程

### 第一步：确定主题和基调

- 确定故事类型（悬疑、情感、职场、家庭等）
- 明确目标读者群体
- 确定故事核心冲突
- 设定情感基调

### 第二步：构建人物

- 主角：背景、性格、欲望、缺陷
- 反派：动机、手段、与主角的关系
- 配角：功能定位、与主线的关联

### 第三步：设计情节

- 开端：如何引入冲突
- 发展：如何升级矛盾
- 高潮：如何解决核心冲突
- 结局：如何收尾并反转

### 第四步：撰写故事

- 按场景分段撰写
- 注重对话和动作描写
- 控制节奏，张弛有度
- 适时埋设伏笔

### 第五步：打磨润色

- 检查逻辑是否自洽
- 强化反转的合理性
- 优化开头和结尾
- 调整语言风格，去除AI味

## 去AI味技巧

1. **具体化描写**：用具体细节代替抽象描述
2. **口语化表达**：适当使用口语、方言、网络用语
3. **情感真实**：让人物情感反应符合人性
4. **不完美**：允许人物犯错，情节有瑕疵
5. **生活气息**：加入生活细节，增强真实感

## 使用场景

- **知乎盐选投稿**：为知乎盐选专栏创作故事
- **短篇小说创作**：创作适合付费阅读的短篇故事
- **故事练习**：练习悬疑反转类故事写作
- **内容变现**：创作可付费的故事内容

## 输出格式

```markdown
# [故事标题]

## 故事大纲
[一句话概括 + 主要情节]

## 人物设定
[主要角色介绍]

## 正文
[完整故事内容]

## 创作说明
[亮点分析、反转解析]
```

## 示例主题

- 职场悬疑：办公室里的秘密
- 情感反转：看似完美的婚姻
- 家庭伦理： inheritance 争夺
- 悬疑推理：密室杀人案
- 社会现实：底层小人物的逆袭
