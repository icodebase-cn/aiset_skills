# 文本处理技巧 (Text Processing Skills)

## 概述
文本处理技巧涉及对小说文本的扩写、总结、重构等操作，帮助作者优化内容表达、调整篇幅和风格。

---

## 1. 文本扩写 (Text Expansion)

### 技巧描述
在保持原文核心情节的基础上，增加细节描述和情感表达，让场景更加生动丰富。

### 核心能力
- **细节丰富**：增加环境、动作、心理等细节描写
- **情节扩展**：在不偏离主线的前提下合理扩展
- **角色深化**：深入刻画角色的心理活动和行为细节
- **环境渲染**：增强场景描写和氛围营造
- **对话优化**：丰富对话内容，增加语言层次

### 扩写原则
- 保持原文的核心情节和人物关系
- 严格按照指定的长度和风格要求执行
- 确保扩写内容与原文风格保持一致
- 让情节发展更加自然流畅
- 避免偏离原文的主要情节线
- 保持故事的连贯性和角色性格的一致性

### 扩写维度
1. **感官细节**：增加视觉、听觉、嗅觉等感官描写
2. **心理活动**：添加角色的内心独白和情感变化
3. **动作细节**：细化动作描写，增强画面感
4. **环境氛围**：扩展环境描写，营造氛围
5. **对话延伸**：丰富对话内容，增加层次感

### 提示词结构
```
## 扩写要求
- **扩写长度**: {{length}}
- **扩写风格**: {{style}}
- **具体指令**: {{instructions}}

## 需要扩写的文本
{{input}}

## 相关上下文
{{context}}
```

## 2. 文本总结 (Text Summary)

### 技巧描述
提炼和总结故事要点，提取关键情节和重要信息，去除冗余细节。

### 核心能力
- **内容提炼**：提取关键情节和重要信息
- **逻辑梳理**：保持总结的准确性和完整性
- **重点突出**：识别并突出重要的故事转折点
- **主题把握**：概括主要主题和情感线索
- **结构优化**：按照要求的详细程度进行总结

### 总结原则
- 准确反映原文的主要内容和情节发展
- 严格按照指定的长度和风格要求执行
- 保持逻辑清晰，条理分明
- 突出关键的情节转折和角色发展
- 保留重要的情感节点和主题元素
- 使用简洁明了的语言表达

### 总结类型
1. **情节总结**：按时间线梳理主要事件
2. **人物总结**：概括角色的发展和变化
3. **主题总结**：提炼作品的核心主题
4. **章节总结**：概括每章的核心内容

### 输出格式
```
## 总结内容
[简洁准确的总结文本]

## 关键要点
- 要点1
- 要点2
- 要点3
```

## 3. 文本重构 (Text Refactor)

### 技巧描述
优化和重构小说文本，改善表达方式，调整语言风格。

### 核心能力
- **文字优化**：改善表达方式，使文字更流畅生动
- **风格调整**：根据要求调整语言风格、叙述角度
- **结构重组**：优化句式结构，改善段落组织
- **细节完善**：补充必要细节，删减冗余内容

### 重构原则
- 保持原文的核心内容、情节发展和人物性格
- 确保与小说整体风格和背景设定保持一致
- 根据上下文信息调整表达方式，保证连贯性
- 尊重作者的创作意图，在此基础上优化
- 严格按照指定的重构方式和长度要求执行

### 重构类型
1. **风格转换**：如从华丽转为简洁，从现代转为古风
2. **视角转换**：如从第一人称转为第三人称
3. **节奏调整**：调整叙述节奏，加快或放慢
4. **情感调整**：调整情感基调，如从悲伤转为希望

## 4. 章节大纲提取 (Outline Extraction)

### 技巧描述
从小说文本中提取章节大纲，分析每章的核心剧情。

### 核心能力
- **情节识别**：识别章节的核心情节线
- **事件提取**：提取关键事件和转折点
- **人物追踪**：概括人物发展轨迹
- **设定标注**：标注重要的设定信息

### 提取原则
- 直接按顺序总结每章的核心剧情
- 简洁明了，不需要过多解释
- 突出情节的主线和关键转折
- 保持章节间的逻辑连贯性

### 输出格式
```
【剧情1】[第一章核心剧情]
【剧情2】[第二章核心剧情]
【剧情3】[第三章核心剧情]
...
```

---

## 5. 场景摘要生成 (Scene Summarization)

### 技巧描述
为小说场景生成简洁准确的摘要，保留核心信息。

### 核心能力
- **关键提取**：识别场景中的核心情节点
- **人物把握**：提取主要角色的关键行为
- **环境概括**：总结环境设定和氛围特点
- **情感捕捉**：概括情感变化和心理转折

### 摘要原则
- 准确捕捉场景的核心内容和主要事件
- 严格按照指定的长度和风格要求执行
- 突出关键角色的重要行为和决定
- 简洁明了，避免冗余和次要细节
- 保留推动故事发展的关键信息
- 体现场景的情感基调和氛围

---

## 6. 摘要扩展为场景 (Summary to Scene)

### 技巧描述
将简洁的情节摘要扩展为生动详细的场景描写。

### 核心能力
- **情节还原**：根据摘要构建完整的场景情节
- **细节创造**：创造丰富的环境描写和氛围
- **对话设计**：设计自然流畅的人物对话
- **心理刻画**：添加恰当的心理描写
- **风格统一**：确保场景与小说整体风格一致

### 扩展原则
- 严格遵循摘要中的核心情节和关键事件
- 严格按照指定的长度和风格要求执行
- 合理扩展细节但不偏离主要故事线
- 创造符合小说风格和时代背景的描写
- 确保人物行为和对话符合其性格特征
- 平衡动作、对话、心理和环境描写

---

## 7. 拆细纲 (Detailed Outline Deconstruction)

### 技巧描述
将章节内容拆解为详细的细纲，按顺序列出每个核心剧情点。

### 核心能力
- **剧情拆解**：将复杂情节拆解为独立剧情点
- **顺序梳理**：按时间顺序排列剧情
- **要点提炼**：提取每个剧情的核心要素
- **逻辑梳理**：保持剧情的逻辑连贯性

### 输出格式
```
【剧情1】[第一个核心剧情点]
【剧情2】[第二个核心剧情点]
【剧情3】[第三个核心剧情点]
...
```

---

## 文本处理通用原则

1. **保持核心**：处理后的文本必须保留原文的核心内容
2. **风格一致**：处理结果要与原文风格保持一致
3. **逻辑连贯**：确保处理后的文本逻辑清晰连贯
4. **尊重原意**：在不改变原意的基础上进行优化
5. **灵活调整**：根据具体要求灵活调整处理方式

---

## 应用场景

- **篇幅调整**：根据需要对文本进行扩写或精简
- **风格统一**：调整不同章节的风格使其统一
- **内容优化**：改善表达，提升文本质量
- **大纲规划**：从已有内容提取大纲或扩展大纲
- **快速预览**：通过摘要快速了解内容
