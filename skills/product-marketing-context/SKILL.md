# 产品营销上下文（product-marketing-context）

## 描述

产品营销上下文 - 创建和维护产品营销基础文档，定义产品定位、目标受众、价值主张和竞争差异。所有其他营销技能的基础，在进行任何营销活动前应先阅读此文档。

## 功能

- 产品定位：明确产品在市场中的位置
- 目标受众定义：识别和描述目标客户群体
- 价值主张提炼：总结产品的核心价值
- 竞争差异分析：分析与竞品的差异化优势
- 营销基础文档：创建可共享的营销上下文文档

## 核心原则

1. **基础性**：这是所有其他营销技能的基础
2. **一致性**：确保所有营销材料传达一致的信息
3. **清晰性**：用简洁明了的语言描述产品和受众
4. **可更新性**：随着产品和市场变化持续更新

## 文档结构

### 1. 产品概述

- 产品名称和一句话描述
- 产品类别和主要功能
- 目标用户群体

### 2. 价值主张

- 主要价值主张（一句话）
- 次要价值主张（3-5条）
- 解决的问题和痛点

### 3. 目标受众

- 主要用户画像
- 用户特征和需求
- 使用场景

### 4. 竞争定位

- 主要竞争对手
- 差异化优势
- 市场定位策略

### 5. 关键信息

- 品牌调性和声音
- 核心关键词
- 禁用词汇

## 使用场景

- **新产品发布**：为新产品建立营销基础
- **品牌重塑**：重新定义产品定位和受众
- **市场扩展**：进入新市场前明确策略
- **团队协作**：确保团队对产品有统一理解
- **代理合作**：向外部合作伙伴传达产品信息
